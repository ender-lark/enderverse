# Plan 04 — Pitch Generator

> Produces a one-page, low-text, high-signal, partly verbatim-speakable pitch per prospect. Source research: `docs/research/03-sales-pitch.md`. Audience: people-skilled callers who are NOT marketers.

## Output: the `Pitch` artifact

Three zones, rendered top→bottom. Generated by **one LLM call** against a strict schema (schema-validated, retry on mismatch), seeded with the prospect record + the competitor record + the computed ROI/price numbers (so the LLM fills language, not math).

```ts
type Pitch = {
  header: { name; category; town; phone; bestCallWindow };       // window default Tue–Thu 10–12 / 2–4 local
  readAloud: string[];        // 5–7 verbatim bullets, in order (see below)
  keyFacts: {
    stars; reviewCount; category;
    competitor: { name; rank; stars; reviews; hasSite: true };
    coreKeyword;                                                  // "[service] near [town]"
    roi: { monthlySearchers; lostPct; lostPerMonth; avgTicket; revenueAtRisk };
    offer: { packageName; oneTime?; monthly; paymentNote };
  };
  deeper: {
    objections: { objection; rebuttal }[];                        // always incl. Busy, Expensive, Facebook
    offerDetail: string;                                          // what's included + timeline + risk-reversal
    nextStepEmail: string;                                        // one-line same-day follow-up
    sourcesLine: string;                                          // 1–2 quotable stats
  };
};
```

### Read-aloud bullets (fixed sequence, bold when rendered)
1. **opener** — name + company + own-the-interrupt ("I know I'm calling out of the blue")
2. **permission** — "Can I borrow 27 seconds…then you decide?"
3. **hook** — their stars/reviews + "nothing to find when people Google you"
4. **competitor_jab** — named competitor ranks #1 despite weaker reviews (only if a qualifying competitor exists)
5. **loss_math** — "you're likely losing ~X bookings/month right when people look you up"
6. **discovery_q** — one light SPIN question
7. **close** — book a 15-min follow-up with **two specific times**

## Inputs the generator needs (all precomputed before the LLM call)

| Input | Source |
|---|---|
| Prospect stars, reviews, category, phone, town | Prospect record (Places) |
| Competitor (name, rank, stars, reviews, has-site) | Competitor finder (Plan 03) — the strongest device when the prospect's reviews are BETTER than the ranked competitor's |
| `coreKeyword` | `"{service} near {town}"` from category + locality |
| ROI math | Prospect-engine WTP/ROI calc (Plan 03): est. monthly searchers × lost% (20–35%) × avg ticket (industry) |
| Offer (package + monthly price) | Pricing engine (Plan 03 / Research 02): BASE × industry × geo × strength |
| Objections | Static library (Busy/Expensive/Facebook + category-specific), lightly personalized by LLM |
| Sources line | Static stat bank (97% search before visiting; 75% judge credibility by website; etc.) |

## Generation rules
- **Every number pre-filled.** The LLM never computes ROI or price — it phrases the prepared numbers. Math lives in code (testable).
- **Competitor framing:** factual, named once, no trash-talk; emphasize the asymmetry ("more reviews than them, but they show up and you don't"). Omit the competitor bullet gracefully if no strong competitor is found.
- **ROI uses defensible ranges, labeled as estimates** (honesty guardrail). Show the assumption (lost% range) so it's not false precision.
- **Tone:** plain, confident, skimmable. Bullets are self-contained spoken lines, not paragraphs.
- **Length cap:** read-aloud ≤ ~7 bullets; deeper section one line per item.

## Rendering
- Print/share-friendly one-pager (the operator may read it on a phone or print it).
- Zone A visually dominant (large text, bold speakable lines); Zone B a scan-in-2-seconds fact box; Zone C compact.
- "Copy script" and "Copy follow-up email" buttons.
- Export to PDF.

## Mock mode
Returns a fully-formed example `Pitch` (the Lumière Med Spa template from Research 03) so the flow renders without an LLM call.

## Acceptance criteria
- Given a prospect + competitor + computed numbers, produces a valid `Pitch` with 5–7 speakable bullets and all key facts filled.
- ROI math matches the prospect-engine output exactly (no LLM arithmetic).
- Reads naturally aloud; a non-marketer could deliver it cold. Objection rebuttals are present and crisp.
